# ATM-Simulation-Project

## Description
A simple ATM simulation program written in C that allows users to
check balance, deposit money, withdraw money, and validates PIN access.

## Features
- PIN validation
- Check account balance
- Deposit and withdraw money
- Menu-driven program

## Technologies Used
- C Programming Language

## How to Run
1. Compile the program using a C compiler
2. Run the executable file
